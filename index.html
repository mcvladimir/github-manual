How to Upload a Website to GitHub (No Coding Needed)

This guide shows exactly where to click to upload a website using only the GitHub website.

What You Need

  • A GitHub account
  • Website files (must be name as index.html)
  • A web browser (Chrome, Edge, Firefox, etc.)

Step 1: Create a New Repository

  1. Go to https://github.com
  2. Log in
  3. Click the ＋ icon (top-right corner)
  4. Select New repository
  5. Enter a repository name (example: my-website)
  6. Select Public
  7. Click Create repository

Step 2: Upload Website Files (Exactly Where to Click)

Case A: Repository is EMPTY (first time)

  1. Open your new repository
  2. Find the blue box called “Quick setup”
  3. Click the blue text “uploading an existing file”

Case B: Repository already has files

  1. Click the Code tab (top of the page)
  2. Click Add file (top-right)
  3. Click Upload files

After clicking Upload:

  1. Drag and drop ALL your website files into the box
  2. Make sure index.html is included
  3. Scroll down
  4. Click Commit changes (green button)

Step 3: Publish Website with GitHub Pages

  1. Click the Settings tab
  2. Click Pages in the left menu
  3. Source: select Main
  4. Folder: select / (root)
  5. Click Save
⏳ Your website may take 1–2 minutes to become available.

Step 4: Open Your Website

Your website URL will look like:

https://your-username.github.io/your-repository-name/

You can find this link in the GitHub Pages section.

✅ Done! Your website is now live and can be shared.
